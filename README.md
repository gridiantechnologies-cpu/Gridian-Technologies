# Gridian Technologies

A premium SaaS Agency website for Gridian Technologies, built with Next.js 14, Tailwind CSS, Framer Motion, and Lucide React.

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS with custom Cobalt Blue theme
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Typography**: Inter (via @fontsource/inter)

## Design System

- Primary Color: Cobalt Blue `#0047AB`
- Background: Pure white `#FFFFFF` / Light gray `#F9F9F9`
- Cards: `rounded-3xl` on all cards
- Glassmorphism: `backdrop-filter: blur(10px)` on nav and glass cards
- Typography: Inter with wide letter-spacing on headers

## Sections

1. **Navbar** - Sticky, glassmorphism on scroll, mobile-responsive
2. **Hero** - Bold headline, CTA buttons, mini bento stats
3. **Services** - 7-card Bento grid layout
4. **About** - Split layout with mission card and founding stats
5. **Stats** - Dark section with company metrics
6. **Testimonials** - 6-card testimonial grid
7. **CTA** - Cobalt gradient CTA with contact info
8. **Footer** - Links, social icons, copyright

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.
